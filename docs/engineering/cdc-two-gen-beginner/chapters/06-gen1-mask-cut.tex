\section{路标规则：低位全 0 就切}

\begin{codebox}[第一代切点条件]
\begin{lstlisting}
if ((h & mask) == 0) {
    在这里切;  // 路标命中
}
\end{lstlisting}
\end{codebox}

\begin{analogybox}[抽奖机]
想象 $h$ 是抽奖号码，mask 规定「末尾若干位必须是 000...0 才算中奖」。  
末尾 8 位全 0 的概率 $\approx 1/256$；末尾 10 位全 0 $\approx 1/1024$。  
\textbf{mask 越「长」（1 越多），切点越稀疏，chunk 平均越大。}
\end{analogybox}

程序根据你配置的 \texttt{avg\_size} 自动算出 mask 里有多少个 1。  
你不需要手算；只要知道：**avg 越大 → mask 越长 → 切得越稀**。

\begin{stepbox}[Step 5 — 第一代切点 = 一个条件]
\[
\text{切} \Leftrightarrow (h \ \& \ \text{mask}) = 0
\]
就这一条（再加上 min/max 护栏）。
\end{stepbox}
