本手册独立描述 ebbackup \textbf{Topo 叙事后三代}内容定义分块（CDC）内核：

\begin{enumerate}[nosep]
  \item \textbf{Gen3 TopoCDC Hom-0}（\featureflag{0x20000} / \envvar{topocdc}）：Gear primary $\wedge$ 滑动 UF $\Delta C$；
  \item \textbf{Gen4 TopoChain}（\featureflag{0x40000} / \envvar{topochain}）：XOR-LFSR stride $\wedge$ 同构 $\hat\beta_0$ $\wedge$ 可选 $\hat\beta_1$，\textbf{Gear-free}；
  \item \textbf{Gen5 TopoPH}（\featureflag{0x80000} / \envvar{topoph}）：Gear primary $\wedge$ Tri-v2 / PH-H0 事件过滤（\textbf{混合态}）；
  \item \textbf{Gen5.1 TopoPH-Native}（\featureflag{0x100000} / \envvar{topophn}）：内容定相 landmark 上的 Tri / PH-H0 \textbf{真切点}，禁止 Gear。
\end{enumerate}

各代 \textbf{独立 dedup 域}，禁止就地互转。Content ID 恒为 \texttt{SHA256(明文 chunk)}，与 CDC 路径无关；但\textbf{切点集合}决定 reuse 与增量带宽。
