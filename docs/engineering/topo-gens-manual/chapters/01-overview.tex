\section{跨代硬约束}

\begin{invariantbox}[五条不可变]
\begin{enumerate}[nosep]
  \item Content ID $=\texttt{SHA256}$（明文）；CDC 只改边界；
  \item 整文件 $\equiv$ 512KB 多 feed cut（parity）；
  \item 各族 dedup 域隔离（跨 flag 复用率 0\%）；
  \item 运行时互斥：gtcdc / topocdc / topochain / topoph / topophn；
  \item 冻结：Chain 切点、Gen5.0 混合切点、FastCDC 默认；Native 仅自树演进。
\end{enumerate}
\end{invariantbox}
