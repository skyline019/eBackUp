\chapter{流式续扫：512KB feed 故事板}
\label{ch:streaming-story}

\begin{analogybox}[接力赛]
整文件 CDC 像一人跑完全程；Streaming 是 512\,KB 一棒。v6 在交棒时必须把\textbf{两路 hash} 和\textbf{下一扫描绝对坐标} 交给下一棒——否则切点会在 feed 边界「漂移」。
\end{analogybox}

\section{Parity 不变量}

\begin{invariantbox}[整文件 $\equiv$ 流式]
对任意文件 $F$，$Offsets(ChunkCuts(F)) = Offsets(Streaming(F))$。这是 \texttt{GtCdc2FTest} 与 Pipeline 集成测试的硬断言。
\end{invariantbox}
