\chapter*{摘要}
\addcontentsline{toc}{chapter}{摘要}
本专册以\textbf{形象化叙事 + 数学推导 + 源码锚点}完整呈现 ebbackup 备份内核的分块子系统：
\begin{enumerate}[nosep]
  \item \textbf{FastCDC}：内容定义分块、Gear rolling hash、mask 期望间距推导；
  \item \textbf{流式 CDC}：32\,MB feed、carry tail、Phase B handoff 与 drift 防护；
  \item \textbf{HCRBO/CFI}：增量备份如何跳过 unchanged chunk；
  \item \textbf{Pipeline 路由}：Inline / Streaming / Hybrid / FastSlice / v4 五路径决策树。
\end{enumerate}
全文含 TikZ 流程图、时序图、对比表与不变量框，供答辩与 code review 使用。
