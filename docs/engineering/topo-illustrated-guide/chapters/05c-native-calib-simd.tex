\section{\texorpdfstring{tau 不进超块}{tau 不进超块}}

\begin{metaphor}[墙上只贴网格纸]
仓里主要保存 stride（和 $k$/persist 位）。\\
$\tau$ 像``今天心情''：用同一把种子 + 墙上数字，每次开房都能精确重算出来。
\end{metaphor}

真名：\texttt{CalibratePhnRuntimeParams}；backup 进程里还可缓存结果。

\section{Default 标定，其它档按比例换衣服}

InitRepo 常在 Default（avg=256KiB）下标定出 $S_{\mathrm{SB}}$。\\
小/大文件：
\begin{equation}
S'=\mathrm{RoundPow2}\big(S_{\mathrm{SB}}\cdot\mathrm{avg}/256\mathrm{KiB}\big).
\end{equation}

\section{Pipeline 三槽冰箱}

\begin{metaphor}[早餐预烤]
Small / Default / Large 各烤一份 runtime 旋钮放进冰箱。\\
同档文件不用每次重新标定 —— \textbf{切点公式不变}，只省火。
\end{metaphor}

\section{4MiB 样音}

Native 标定样本更大（4MiB），因为事件更稀、更噪。\\
目标平均块长仍在 $[0.85,1.15]\times\mathrm{avg}$。

\section{吞吐直觉}

几乎每字节炼 $Y$（比 Chain 拧锁贵）—— 所以 vs Stream 门禁更松（$\ge0.80$），允许更高的 ns/probe。\\
这不是``没调好''，是去掉指纹铃后的结构税。
