典型 Fail 剧情：
\begin{itemize}[nosep]
  \item Hom 仓却没喊 \texttt{topocdc}；
  \item Fast 仓已有库存，却突然喊 Topo 口令；
  \item Chain $L$ 不在 8..22；MVP 仓想就地开环检；
  \item Native Tri 仓改跑 PH；
  \item 同进程多口令并存。
\end{itemize}
