\begin{v6box}[本手册写什么]
\textbf{G-TCDC v6 2F-Gear}（superblock flag \texttt{0x1000|0x10000}）是 ebbackup 2026-07 起的\textbf{生产默认} G-TCDC 内核。本手册\textbf{只写 v6 相对基线的增量}：双指纹、非重叠 bit 域 mask 拆分、流式 resume、引擎接线与 bench 门禁。
\end{v6box}

  \item Pipeline v4 阶段划分、worker 并行：\crossrefmanual{chapters/12-pipeline-v4.tex}
  \item Streaming 32\,MB feed / carry tail 框架：\crossrefmanual{chapters/13-streaming-cdc.tex}
  \item HCRBO/CFI 增量语义：\crossrefmanual{chapters/06-chunk-hcrbo.tex}
  \item Rabin v2 张量路径（兼容仓）：\filepath{docs/engineering/gtcdc-proof/}
\end{itemize}
\end{designbox}
